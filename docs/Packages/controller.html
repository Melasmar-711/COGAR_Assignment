Trajectory Manager

Trajectory Manager Node.

This node manages robot trajectories by coordinating between action planning, pose reachability checking, and PID control. It receives target positions, verifies reachability, and sends trajectories to the PID controller.

ROS Interfaces:

Services:

  • /get_trajectory (assignments.srv.CheckJointState)

Manages robot trajectory execution through coordination with other nodes.

PID Controller

PID Controller Node.

This node implements a dummy PID controller that simulates joint position control. It receives trajectory requests and publishes dummy control outputs while monitoring joint state errors.
