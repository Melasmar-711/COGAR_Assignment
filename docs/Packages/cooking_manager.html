Human Command Manager

Human Command Manager Node.

This node manages human verbal commands by validating them against system state and current action sequences. It coordinates between verbal commands, system state, and action sequences to determine when commands can be safely executed.

Action Planner

Action Planner Node.

This node translates recipe steps into actionable sequences and manages their execution. It handles both automated recipe steps and human intervention commands, coordinating between the recipe tracker, command monitor, and controller nodes.
